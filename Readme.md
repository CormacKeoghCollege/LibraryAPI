# Library Management REST API

A complete serverless REST API built with .NET 8, AWS Lambda, and PostgreSQL.

## Quick Start
1. Clone this repository
2. Follow the [Tutorial Guide] TutorialInstructions.txt
3. Deploy to AWS using SAM

## Architecture
- AWS Lambda + API Gateway
- PostgreSQL RDS Database  
- AWS Secrets Manager
- JWT Authentication
